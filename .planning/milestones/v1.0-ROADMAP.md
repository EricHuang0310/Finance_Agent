## Key Decisions

| Decision | Rationale |
|----------|-----------|
| CIO as Team Lead | Avoids extra Opus teammate cost; CIO naturally orchestrates |
| Phased teammate spawning | Keeps active count at 3-5, saves tokens |
| Sector Specialist as context enrichment, not 4th debater | Avoids 15-30s latency penalty |
| Execution Strategist as code, not LLM | Order type selection is quantitative |
| Pattern learning via tag grouping, not clustering | Low trade volume makes clustering impractical |
