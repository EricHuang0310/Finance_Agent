       ```python
       JOURNAL_PATH = Path("logs/trade_journal.json")
       JOURNAL_LOCK = Path("logs/trade_journal.json.lock")
       ```

       Functions:

       **`journal_on_fill(trade: dict, order_result: dict) -> None`** (per D-04):
       Build journal entry dict:
       - `order_id`: order_result["id"]
       - `symbol`: trade["symbol"]
       - `side`: trade["side"]
       - `qty`: trade["suggested_qty"]
       - `entry_price`: trade["entry_price"]
       - `stop_loss`: trade.get("stop_loss")
       - `take_profit`: trade.get("take_profit")
       - `initial_risk`: abs(trade["entry_price"] - trade["stop_loss"]) if trade.get("stop_loss") else None
       - `entry_thesis`: dict with composite_score, sector, signals_at_entry (from trade.get("signals_summary", {})), cio_stance (from trade.get("risk_assessment", {}).get("cio_stance")), portfolio_correlation (from trade.get("portfolio_correlation"))
       - `filled_at`: datetime.now().isoformat()
       - `status`: "open"
       Call `_append_journal(entry)`.

       **`journal_on_close(candidate: dict, order_result: dict) -> None`** (per D-04):
       - Load journal via `_load_journal()`
       - Search REVERSED for matching entry: same symbol AND status=="open"
       - Compute: entry_price, exit_price=candidate["current_price"], direction (1 if side=="buy" else -1), pnl_per_share, pnl, pnl_pct, r_multiple (pnl_per_share / initial_risk if available)
       - Update the entry dict (in-place mutation is acceptable here since we're writing back the whole list):
         - exit_price, exit_reason, closed_at, pnl (round 2), pnl_pct (round 4), r_multiple (round 2 or None), outcome (_classify_outcome), holding_days, status="closed", close_order_id
       - Save via `_save_journal(journal)`
       - If no matching open entry found, print a warning and skip (do not crash)

       **`_classify_outcome(pnl_pct: float) -> str`** (per D-05):
       - |pnl_pct| < 0.005 -> "scratch"
       - pnl_pct > 0 -> "win"
       - else -> "loss"

       **`_compute_holding_days(filled_at: str, closed_at: str) -> int`**:
       - Parse ISO timestamps, return difference in days

       **`_load_journal() -> list[dict]`**:
       - If JOURNAL_PATH exists, read and return list
       - Else return []
       - Wrapped in FileLock(JOURNAL_LOCK, timeout=10)

       **`_save_journal(journal: list[dict]) -> None`**:
       - Write via save_state_atomic
       - Wrapped in FileLock(JOURNAL_LOCK, timeout=10)

       **`_append_journal(entry: dict) -> None`**:
       - Load, append, save (all under FileLock)

       Imports needed: Path, datetime, json, FileLock from filelock, save_state_atomic from src.utils.state_io
